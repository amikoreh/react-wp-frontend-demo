React WP Frontend demo
================
### Wordpress REST API + React JS
#### Playing with React JS and WordPress

#### Application Requires/Configuration:
* React JS 15.4
* Node v6.11.3
* npm 5.4.2

##### Build + Install:
* `npm install -g create-react-app`
* `cd react-wp-frontend-demo`
* `npm install`
* `npm test`
* `npm start`
* `npm run build`
